# 记忆桌宠网页共读扩展

这是无需商店发布即可加载的 Chrome / Edge Manifest V3 扩展。网页正文只在用户点击右键菜单时读取并通过带随机配对令牌的 `127.0.0.1` 本机桥接交给桌宠。用户还可以另行开启“接收桌宠中已审批的网页操作”，让扩展领取已经确认的一次性刷新、前进/后退、向上/下滚动、滚到顶部/底部或页内查找命令。

## 本地加载

1. 启动桌宠，在“设置 → 电脑协作”中开启“电脑协作”和“浏览器右键共读桥接”。
2. 点击“打开扩展目录”。
3. Chrome 打开 `chrome://extensions`，Edge 打开 `edge://extensions`；开启开发者模式，选择“加载已解压的扩展程序”，指向本目录。
4. 在桌宠设置中点击“复制配对信息”，再点击浏览器工具栏里的扩展图标，粘贴并测试。
5. 需要当前页操作时，勾选“接收桌宠中已审批的网页操作”并保存。扩展使用 `activeTab` 临时权限；导航或权限失效后，请在目标网页再点击一次扩展图标。

令牌允许提交用户主动共享的文本，以及领取桌宠界面中已经确认的一次性固定命令；接口不返回记忆、设置、本机文件或页面数据。即使该命令属于 2～4 步协作计划，扩展也只会在前一步完成且下一步再次得到确认后领取一项。命令 45 秒过期、结果只提交一次，扩展不接受通用脚本或任意点击。重新生成令牌后，旧令牌立即失效。
