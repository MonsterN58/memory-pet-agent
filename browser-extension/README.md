# 记忆桌宠网页共读扩展

这是无需商店发布即可加载的 Chrome / Edge Manifest V3 扩展。它只在用户点击网页右键菜单时，读取当前选中文本或当前页经过裁剪的正文，并通过带随机配对令牌的 `127.0.0.1` 本机桥接交给桌宠。

## 本地加载

1. 启动桌宠，在“设置 → 电脑协作”中开启“电脑协作”和“浏览器右键共读桥接”。
2. 点击“打开扩展目录”。
3. Chrome 打开 `chrome://extensions`，Edge 打开 `edge://extensions`；开启开发者模式，选择“加载已解压的扩展程序”，指向本目录。
4. 在桌宠设置中点击“复制配对信息”，再点击浏览器工具栏里的扩展图标，粘贴并测试。

令牌只赋予“向桌宠提交一段用户主动共享的文本”这一项能力；接口不返回记忆、设置或本机文件。重新生成令牌后，旧令牌立即失效。
